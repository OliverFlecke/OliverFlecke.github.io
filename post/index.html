  1. 2021-08-11 | Authenticate with Github on a Single-Page Application

    Introduction For a recent project, I wanted a way to authenticate users, but without all the struggle of managing users and their data. This was for a frontend, single-page application (SPA) where everything could be served as static HTML/CSS/JS files. My solution was to allow users to sign in with their Github credentials using OAuth. This is the best of both worlds: I won’t have to store any user data or think much about security, and users don’t have to trust me with their email or password.
